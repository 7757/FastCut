全局快捷键

任意界面按 ⌘⇧V 唤起历史,快捷键可在偏好设置里自定义。

智能类型识别

链接、邮箱、文件路径、颜色值、数字自动分辨,一眼看清。

键盘全操作

输入即搜索,方向键选择,回车粘贴 —— 双手不离键盘。

自动回贴

选中即自动粘回你刚才所在的应用(需辅助功能权限)。

文本 · 图片 · 文件
