功能

小而快,恰到好处

该有的一个不少,多余的一个不加。对标市面主流剪贴板工具,只留最常用的。

全局快捷键

任意界面按 ⌘⇧V 唤起历史,快捷键可在偏好设置里自定义。

智能类型识别

链接、邮箱、文件路径、颜色值、数字自动分辨,一眼看清。

键盘全操作

输入即搜索,方向键选择,回车粘贴 —— 双手不离键盘。

自动回贴

选中即自动粘回你刚才所在的应用(需辅助功能权限)。

文本 · 图片 · 文件

复制什么都记得住,图片和复制的文件路径也一并保存。

隐私优先

自动忽略密码管理器标记的敏感内容,历史只存在你本机。

原生轻量

纯 Swift 打造,零第三方依赖,常驻菜单栏,几乎不占资源。

开源免费

MIT 协议,代码完全公开,欢迎 Star、提 Issue 或自己动手改。

键盘驱动

双手不离键盘

唤起、搜索、选择、粘贴、删除,一整套流程都能用键盘完成。

↑↓选择
