# FastCut

[English](README.md) · [简体中文](README.zh-CN.md) · **日本語** · [한국어](README.ko.md)

軽量なネイティブ **macOS クリップボード履歴マネージャー** — Maccy / Flycut のようなツールの、
シンプルな代替。メニューバーに常駐し、コピーした内容を記憶して、グローバルホットキーで履歴を呼び出します。

**🌐 [ウェブサイト](https://7757.github.io/FastCut/) · ⬇️ [ダウンロード](https://github.com/7757/FastCut/releases/latest)**

![FastCut クリップボードのポップアップ](docs/popup.png)

## ⚡ インストール

**ワンライナーでインストール** — 最新版をダウンロードして `/Applications` に配置し、起動します:

```sh
curl -fsSL https://7757.github.io/FastCut/install.sh | bash
```

**Homebrew:**

```sh
brew install --cask 7757/fastcut/fastcut
```

**手動** — [Releases](https://github.com/7757/FastCut/releases/latest) から最新の `.app` を入手し、
`/Applications` へ移動、初回起動は **右クリック → 開く**(自己署名・未公証のため）。

**ソースからビルド** — 下の [ビルド](#ビルド) を参照。

## 機能

- **クリップボード履歴**:テキスト、画像、コピーしたファイルパス
- **グローバルホットキー**(既定 **⌘⇧V**)で検索可能な履歴ポップアップを表示 — 設定で**カスタマイズ**可能
- **キーボード操作**:入力で検索、`↑`/`↓` で移動、`↩` で貼り付け、`⌘⇧⌫` で削除、`⎋` で閉じる
- **自動貼り付け**:直前に使っていたアプリへそのまま貼り付け(アクセシビリティ権限が必要)
- **メニューバー**から最近の 8 件へすばやくアクセス
- **プライバシー重視**:パスワードマネージャーが付ける機密/一時的な項目は自動的に無視
- **永続化**;履歴件数は調整可能;**ログイン時に起動**(任意)
- **自動アップデート確認**:新しいバージョンをメニューバーで通知
- メニューバーのみ(Dock アイコンなし)、サードパーティ依存ゼロ

## 動作環境

- macOS 14 以降(macOS 26 / Apple Silicon でビルド・テスト)
- Xcode Command Line Tools(`xcode-select --install`)— 完全な Xcode は不要

## ビルド

```sh
./build.sh
```

`swiftc` で Swift ソースをコンパイルし、`FastCut.app` を組み立てて署名します。

> **ヒント — 再ビルド後もアクセシビリティ権限を保持する。** 既定は **ad-hoc** 署名で、その識別子は
> ビルドごとに変わるため、macOS は毎回再許可を求めます。`Assets/setup-signing.sh` を一度実行して
> 安定した自己署名 ID を作成すると、`build.sh` が自動的にそれで署名し、許可が保持されます。
> (証明書はローカル専用で、セキュリティ上の価値はありません。)

## 実行

```sh
open FastCut.app
```

または `FastCut.app` を `/Applications` に入れてそこから起動(推奨)。メニューバーに稲妻アイコンが表示され、
どこでも **⌘⇧V** で履歴を開けます。

### 権限

- **クリップボードの読み取り**と**グローバルホットキー**は権限不要。
- **自動貼り付け**は ⌘V をシミュレートするため、**アクセシビリティ**権限が必要:
  システム設定 → プライバシーとセキュリティ → アクセシビリティ → **FastCut** を有効化。
  未許可の間は、選択するとクリップボードにコピーされるので手動で貼り付けできます。

## アンインストール

メニューバーから終了し、`FastCut.app` を削除、`~/Library/Application Support/FastCut/` を削除します。
